製薬会社のアニュアルレポート　表紙・中面
自主制作
概要

ある企業の採用に応募した際に課題として制作した、製薬会社のアニュアルレポートの表紙（2案）および中面デザインです。 課題として提示されたテーマ「New Value Creation（新しい価値の創造）」をもとに、企業が自らの手で未来を描く姿勢を表現しています。 制作期間は合わせて12時間程度。Adobe IllustratorとPhotoshopを使用し、印刷物としての完成度を意識したレイアウトを行いました。

制作にあたって意識したこと

コーポレートコピー「New Value Creation」を、企業の理念や事業内容と一貫性をもって表現することを意識しました。 ステークホルダーとしてまた、「患者」「研究者」「医療従事者」の3者を視覚的に示しながら、堅実さと成長性の両立を感じさせるデザインにしたいと考えました。 企業広報誌として信頼性を保ちつつ、見る人にポジティブな印象を与えるビジュアル設計が課題でした。

制作のコンセプト

「価値を創造する＝自らの手で描く」という発想から、“描く手元”をキービジュアルに設定。 そこに「患者」「研究者」「医療従事者」の3者を象徴する写真を組み合わせ、企業の取り組みが多角的に展開していることを表現することにしました。 表紙についてはスタンダードなデザインとして写真をメインにしたものと、それとは別角度からの案としてイラストを使用した案の2つを作成することに。 中面については、写真の表紙案と同様のイメージでデザインすることにしました。

デザインのポイント

写真をメインにした案では、三角形のモチーフを安定と発展の象徴として多用し、色や大きさの変化で多様な取り組みを暗示しました。 イラスト案では、「描く手元」から連想した一筆書きの自作イラストを使用。 「描く手元」「研究者」「患者」「医療従事者」に加え、薬のモチーフを取り入れることで、製薬企業らしい親しみと創造性を両立しています。 中面デザインでは、社長インタビューのテーマ「これまでを踏まえ、今後をどうしていくか」に合わせ、“歩き出す姿”を象徴する写真を採用。 誌面全体を通して青系のトーンを基調に、理知的で統一感のあるビジュアルにまとめています。
